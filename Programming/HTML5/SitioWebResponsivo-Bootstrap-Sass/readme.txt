Definir objetivos del sitio web:

Objetivo: Este sitio web muestra una interfaz donde se ven y controlan 
datos (operaciones). Consta de 3 páginas:

El público objetivo puede ser un jefe.

1-Página de inicio donde se incluye una barra de navegación fija en la 
parte superior con enlaces a las demás páginas. Se utiliza un carousel 
para mostrar fotos destacadas y un jumbotron para el mensaje principal.

2-Página de servicios donde se organiza la información en un grid de 
Bootstrap y utiliza cards para mostrar cada servicio.

3-Página de contacto utiliza formularios de Bootstrap con campos para 
el nombre, correo electrónico, asunto y mensaje.

Se crea un archivo sass para el sitio web donde se definen variables sass
para colores, fuente y tamaño de texto.
Se utiliza anidación de sass y herencia. El diseño es responsivo
